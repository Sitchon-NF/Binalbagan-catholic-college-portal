# Binalbagan-catholic-college-portal

A simple single-page school website project created using HTML, CSS, and JavaScript.

## Features
- Home Page
- About Page
- Courses Section
- Teachers Section
- Department Section
- Contact Section
- Responsive Navigation
- Dynamic Page Switching using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
```bash
project/
│── index.html
│── style.css
│── script.js
│── Image/
```

## How to Run
1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
